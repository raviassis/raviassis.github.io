Olá pessoal. Sejam bem vindos.

Hoje vou apresentar a vocês algumas formas para converter dados em formato JSON para CSV.

Para quem já trabalhou com banco de dados não relacionais, provavelmente já se esbarrou com esse problema. Trabalhei num projeto que utilizava o banco de dados CosmosDb, e depois de publicar a primeira versão do projeto, começaram a surgir demandas do cliente de gerar alguns relatórios esporádicos.

Depois de extrair os dados do Cosmos, deparavamos com o problema do formato. O cliente esperavam o relatório num formato tabular, como um arquivo excel. Não podia simplesmente entregar os dados daquela forma.

Como converter JSON em CSV

Vou listar abaixo algunas formas para se converter dados JSON em CSV:

Usar uma ferramenta de conversão online

O primeiro lugar que recorri https://json-csv.com/. Resolveu bem o problema. Porem possui um limite de conversão, e para continuar convertendo precisa contratar a versão PRO, que é paga. Já tive que converter arquivos com mais de 100 mil registros.

Pacote npm json2csv

Procurando por uma forma de converter sem limites, encontrei esse pacote npm json2csv. Rápido de se instalar e fácil de se converter. Mas tem como pré requisito ter Node e NPM instalados.

Instalação 

    npm install json2csv -g

Converter arquivo 

    json2csv -i relatorio.json -o relatorio.csv

Por fim. Ferramenta própria

Aproveitando a demanda por esse tipo de conversão, criei uma ferramenta própria para facilitar o trabalho. Json To Csv.
