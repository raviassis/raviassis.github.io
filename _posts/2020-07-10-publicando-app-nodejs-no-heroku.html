Olá pessoal. Hoje vou mostrar como publicar uma aplicação NodeJs no Heroku. Mas o que é cada uma dessas tecnologias?

NodeJs

NodeJs é um ambiente de execução JavaScript desenvolvido com a engine V8 do Chrome.

Veja também: Desenvolvendo uma API com NodeJs e Express

Heroku

O Heroku é uma plataforma em nuvem como serviço (PaaS) baseada em containers. Ela permite que os desenvolvedores publiquem, gerenciem, e escalem suas aplicações. Suporta oficialmente diversas linguagem de programação, como NodeJs, Ruby, Java, PHP, Python, Go, Scala, além de permitir o uso de outras linguagens através de serviços de terceiros.

Para nosso exemplo de hoje, vamos precisar criar uma conta no Heroku e instalar o Heroku CLI.

Crie uma conta no Heroku

Acesse o link https://signup.heroku.com/ e crie uma nova conta.

Instale o Heroku CLI

Acesse o link https://devcenter.heroku.com/articles/heroku-cli baixe e instale o Heroku CLI.

Feito os passos, vamos criar nossa aplicação node.

Criando aplicação Node

Vamos utilizar o Express Generator para gerar uma aplicação node básica. 

$ npm install express-generator -g
$ express deploy-node-heroku
$ cd deploy-node-heroku

Instale as dependências. 

$ npm install

Teste a aplicação local. 

$ npm start

Acesse http://localhost:3000. Veremos a página abaixo.

Pronto, agora podemos publicar nossa aplicação.

Publicando no Heroku

Inicialize um repositório git no diretório da aplicação e dê o primeiro commit. 

$ git init
$ git add .
$ git commit -m “first commit”

Faça login no Heroku utilizando o Heroku CLI. 

$ heroku login

Aparecerá a seguinte mensagem: 

    heroku: Press any key to open up the browser to login or q to exit:

Pressione qualquer tecla para continuar.

O navegador será aberto numa página de login do heroku. Faça o login e volte para o terminal.

Feito o login, vamos criar um app no heroku. 

$ heroku create

O comando acima irá criar um novo container de aplicação e configurar o repositório remoto.

Agora vamos publicar a aplicação. 

$ git push heroku master

Ao finalizar a publicação, execute: 

$ heroku open

O navegador será aberto na url em que nossa aplicação foi publicada.

Pronto. Nossa aplicação está publicada e disponível na web. Espero que tenham gostado. Até a próxima.
